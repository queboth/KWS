iui.core.KWSBatch module¶

This is the new Keyword Spotting Tool, which spot’s search terms in the PyLaia ConfMats.ark.

This is the CLI Batch Module to start a spotting run on mutiple confMats.ark. For a list of possible arguments please start the Programm with the -h argument: python -m iui.core.KWSBatch -h
