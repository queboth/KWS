iui.cli.KWS module¶

This is the new Keyword Spotting Tool, which spot’s search terms in the PyLaia ConfMats.ark.

This is the CLI Module to start a spotting run on a single confMats.ark. For a list of possible arguments please start the Programm with the -h argument: python -m iui.cli.KWS -h
